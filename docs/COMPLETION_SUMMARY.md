## 🚀 How to Start (3 Steps)

### Step 1: Configure
```bash

GROQ_API_KEY=your_key_from_console.groq.com
```

### Step 2: Start
```bash
# Terminal 1: Backend
cd backend
python -m uvicorn main:app --reload --port 8000

# Terminal 2: Frontend
npm run dev
```

### Step 3: Use
