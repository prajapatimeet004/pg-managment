## ⚡ Quick Start (2 minutes)

### Prerequisites Check
1. **Install Python 3.11+** → https://python.org
2. **Install Node.js 18+** → https://nodejs.org
3. Add both to your system PATH

### Run Setup
```bash
# Just double-click this file:
VERIFY_AND_START.bat

# Or run these commands:
python verify_ai_agent.py
```

### Start the System

**Terminal 1 - Backend (Run these commands):**
```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```

**Terminal 2 - Frontend (Run these commands in new terminal):**
```bash
npm run dev
```

**Browser:**
```
Open: http://localhost:5173
Click: "AI Assistant" tab
Start: Asking questions!
```

## 🆘 Troubleshooting

### "Python not found"
- Install Python: https://python.org
- Make sure to check "Add Python to PATH"
- Restart your computer

### "Node not found"
- Install Node.js: https://nodejs.org
- Includes npm automatically
- Restart your computer

### "Port 8000 already in use"
- Change port in command: `--port 8001`
- Or kill the process using port 8000

### "GROQ_API_KEY error"
- Check backend/.env file exists
- It should have GROQ_API_KEY set
- File is automatically created by setup scripts
